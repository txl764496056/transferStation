box-shadow增加或减少投影面积-第4个参数

（可以利用第4个参数设置边框效果）

x轴5px，模糊距离5px
box-shadow:
10px 0 5px #000;
y轴5px，模糊距离5px
box-shadow:
0 10px 5px #000;
内阴影，模糊距离10px
box-shadow:
inset 0 0 10px #000;
内阴影，模糊距离20px
box-shadow:
inset 0 0 20px #000;
阴影面积 0px 模糊距离0px
box-shadow:0 0 20px #000;
或者
box-shadow:0 0 20px 0 #000;
阴影面积 10px 模糊距离20px
box-shadow:0 0 20px 10px #000;
阴影面积 10px，模糊距离 0px
box-shadow:0 0 0 10px #000;
阴影面积 20px，模糊距离 0px
box-shadow:0 0 0 20px #000;
外侧阴影和内侧阴影面积：10px
模糊距离：0px
虚线边框3px

多层阴影支持逗号分隔语法，可以创建任意数量投影

2层 外 测阴影
box-shadow:
0 0 0 10px #ce7c1c,0 0 0 20px #158a8d;
2层 外 测阴影
x、y轴偏移(5px,5px)(-5px,-5px)
box-shadow:
5px 5px 0 5px #ce7c1c,
-5px -5px 0 5px #158a8d;
2层 内 测阴影
box-shadow:
inset 0 0 0 10px #faca93,
inset 0 0 0 20px #a3e4e6;

注意

  • 阴影不会受box-sizing属性的影响。
  • 不会相应鼠标事件，比如悬停或点击。解决办法：加上inset利用内侧阴影

参数：

box-shadow: h-shadow v-shadow blur spread color inset;
h-shadow 必需的。水平阴影的位置。允许负值
v-shadow 必需的。垂直阴影的位置。允许负值
blur 可选。模糊距离
spread 可选。阴影的大小
color 可选。阴影的颜色
inset 可选。内侧阴影,不写inset则是外侧阴影
