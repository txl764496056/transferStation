图片染色效果

方法一：滤镜

  • 原始图片
  • filter:sepia(100%);
    降低饱和度,增加橙黄色染色效果
  • filter:sepia(100%) saturate(4);
    降低饱和度>增高饱和度，增加暖金色染色效果
  • filter:sepia(100%) saturate(4) hue-rotate(-70deg);
    降低饱和度>增高饱和度>旋转色环，改变色相，呈现亮粉色效果。

方法二：混合模式

2个属性:

mix-blend-mode：可以为整个元素设置混合模式。
  要求：需要把图片包裹在一个容器中，并把容器的背景色设置为我们想要的主色调。

background-blend-mode：可以为每层背景单指定混合模式。
   要求：不用图片元素，而是用<div>元素——把这个元素的第一层背景设置为要染色的图片，并把第二层的背景设置为我们想要的主色调。（也就是图片作为背景图）

查看所有混合属性值在后面列出来了

  • 原始图片
  • mix-blend-mode:luminosity;混合
  • background-blend-mode:luminosity;混合
    有没有发现是从绿色到亮粉色的，因为用的透明色到亮粉色进行的混合

滤镜与混合模式的动画效果

滤镜可动画，但是混合模式不行

混合模式虽然不可以，但是我们可以利用背景色模仿动画效果

  • filter进行动画,transition:filter 1s;
  • transition:background-color 1s;纯色混合动画

混合模式类型：


normal:正常 multiply:正片叠底 screen:滤色 overlay:叠加 darken:变暗 lighten:变亮 color-dodge:颜色减淡 hard-light:强光 soft-light:柔光 difference:差值 exclusion:排除 hue:色相 saturation:饱和度 color:颜色 luminosity:亮度 initial:初始 inherit:继承 unset:复原

参数：

filter: none | blur( ) | brightness( ) | contrast( ) | drop-shadow( ) | grayscale( ) | hue-rotate( ) | invert( ) | opacity( ) | saturate( ) | sepia( ) | url( );
none 默认值，没有效果。
blur(px) 给图像设置高斯模糊。值越大越模糊；
如果没有设定值，则默认是0；这个参数可设置css长度值，但不接受百分比值。
brightness(%) 给图片应用一种线性乘法，使其看起来更亮或更暗。如果值是0%，图像会全黑。值是100%，则图像无变化。其他的值对应线性乘数效果。值超过100%也是可以的，图像会比原来更亮。如果没有设定值，默认是1。
drop-shadow(h-shadow v-shadow blur spread color) h-shadow:y轴偏移量。 v-shadow:x轴偏移量。 这两个都接受正负值。
blur:模糊半径。 值越大，越模糊，则阴影会变得更大更淡。不允许负值，默认是0 (则阴影的边界很锐利)。
spread:这是第四个 值。正值会使阴影扩张和变大，负值会是阴影缩小.若未设定，默认是0 (阴影会与元素一样大小)。注意: Webkit, 以及一些其他浏览器 不支持第四个长度，如果加了也不会渲染。但是box-shadow这个属性值有效果
color:阴影颜色值
grayscale(%) 将图像转换为灰度图像。值定义转换的比例。值为100%则完全转为灰度图像，值为0%图像无变化。值在0%到100%之间，则是效果的线性乘子。若未设置，值默认是0；
hue-rotate(deg) 给图像应用色相旋转。"angle"一值设定图像会被调整的色环角度值。值为0deg，则图像无变化。若值未设置，默认值是0deg。该值虽然没有最大值，超过360deg的值相当于又绕一圈。
改变成想要的颜色，参考色环图，决定旋转角度。
invert(%) 反转输入图像。值定义转换的比例。100%的价值是完全反转。值为0%则图像无变化。值在0%和100%之间，则是效果的线性乘子。 若值未设置，值默认是0。
opacity(%) 转化图像的透明程度。值定义转换的比例。值为0%则是完全透明，值为100%则图像无变化。
saturate(%) 转换图像饱和度。接受百分值和具体数值。值定义转换的比例。值为0%则是完全不饱和，值为100%则图像无变化。
sepia(%) 将图像转换为深褐色。值定义转换的比例。值为100%则完全是深褐色的，值为0%图像无变化。值在0%到100%之间，则是效果的线性乘子。若未设置，值默认是0；
url() URL函数接受一个XML文件，该文件设置了 一个SVG滤镜，且可以包含一个锚点来指定一个具体的滤镜元素。例如：filter: url(svg-url#element-id)
initial 设置属性为默认值。
inherit 从父元素继承该属性。
