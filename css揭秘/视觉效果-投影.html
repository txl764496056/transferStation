视觉效果-投影

单侧投影（只有一边有）

利用负数（投影的第4个参数-扩张投影）

投影参数：扩张投影，设为负数，是投影缩短，

模糊参数：设为正数是扩大投影

底部投影
右侧投影
顶部侧投影

领边投影

2个领边投影位移与模糊大小一样，扩展值不一样

领边投影,扩展值是0，显示出来的阴影就是6px
领边投影，扩展值是-3px，显示出来的阴影就是3px

双侧投影

用两块投影，类似把单侧投影用2次

3px两侧投影
6px两侧投影，改变x轴的位移即可

不规则投影

利用css滤镜，filter:drap-shadow(),相关滤镜还有很多，这里不坐描述

drap-shadow()参数：offsetX offsetY r color
offsetX：x轴偏移量 offsetY：y轴偏移量 r：模糊半径 color：阴影颜色 没有扩张半径和inset

圆角效果,投影没问题。box-shadow

四个角本不该有投影。box-shadow
右侧箭头没有投影。box-shadow
投影是矩形,应该是每个实色点有投影才对。box-shadow

4个角不会出现白色和直角投影。filter:drop-shadow( )
右侧箭头有投影了。filter:drop-shadow( )
每个实色点有投影，并且不会出现背景变成白色。注意，文字也应为drop-shadow有了投影。filter:drop-shadow( )
text-shadow与drop-shadow同时存在
